# Overview

## Product Name

Kenpin Studio

## Purpose

画像AIモデル（LoRA / fine-tune）の運用をチームで安全に管理するためのダッシュボード。

## Core Features

* **モデル管理** — LoRA / fine-tune / 推論向けモデルを整理・管理
* **ジョブ監視** — GPU キュー、進捗、レビュー待ち状態を継続監視
* **チーム認証** — NextAuth.js によるセキュアな認証・アクセス制御
* **データセット管理** — データセット品質チェックと一元管理
* **画像DB管理** — 複数の画像DB接続（ローカル/リモート）を登録・管理
* **ワークスペーススタジオ** — 画像前処理 → アノテーション → 学習 → 結果確認の一連フロー
  * **前処理** — リサイズ・切り抜き・グレースケール・二値化・色調調整など Canvas API ベースのプレビュー付き処理
  * **アノテーター** — Konva.js ベースの BBox アノテーション（クラスラベル管理、YOLO エクスポート）
  * **YOLO エクスポート** — `classes.txt` + 画像ごとの `.txt` を出力フォルダへ書き出し
  * **AI 学習** — FastAPI + Ultralytics YOLO によるモデル学習（非同期ジョブ、エポック進捗リアルタイム表示、CUDA 自動検出）

## Target Users

* MLエンジニア
* MLOps担当
* データ管理者

## Value

* 運用の可視化
* チームでの安全な管理
* 前処理〜アノテーション〜学習の一元化による作業効率化
